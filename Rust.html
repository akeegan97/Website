Rust Projects

Below are some of the projects I completed in Rust.

Geometric Brownian Motion in Rust

This project used EGUI to construct a Graphical User Interface that a user can use to drop in Yahoo historical prices and run a GBM simulation on. I previously wrote a Python version of this project with no GUI, iterating on the idea with Rust I was able to expand the program for ease of use and the number of calculations done with Rust.

Investment Simulator

This project used EGUI to construct a Graphical User Interface that a potential investor could use to model their investments with. It included a selection of different investment opportunities as well as parameters than can be changed and graphs where the results are displayed. Finally, users can download the results to a csv file for further understanding or data analytics.
